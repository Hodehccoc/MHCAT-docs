# 如何幫助文檔進行貢獻

<head>
  <title>如何幫助文檔進行貢獻</title>
</head>

## 為何需要貢獻?

一個人要維護MHCAT是非常困難的任務，這並不簡單，尤其開發者還是一名學生，要進行這些事情其實時間非常有限

因此希望可以靠各位的幫助，來幫忙進行一些簡易的文檔編寫

## 我可以獲得甚麼好處 ?

一但您通過驗證，您可以前往官方[discord群](https://discord.gg/7g7VE2Sqna)尋找客服，索取0.5美元的chatgpt儲值金

儲值金可以讓您在discord上與chatgpt(非正式版本)進行聊天

## 我該如何進行貢獻?

* Step 1

要進行貢獻其實非常簡單，每個文檔下方都會有一個 `Edit this page`點進去後會到達github頁面

![](https://media.discordapp.net/attachments/991337796960784424/1080425372446887976/image.png)

* Step 2

如果你沒有guthub帳號的話，請先創建一個

* Step 3

現在你可以開始進行編寫了!

## 編寫原則

基本上你只要知道以下規則

* `## title`為標題

* `### titlel`為小標題

接著有以下幾個標註可以使用

```
:::danger 注意
內容
:::
```
出來的樣式:
:::danger 注意
內容
:::

```
:::tip 訣竅
內容
:::
```
出來的樣式:
:::tip 訣竅
內容
:::
```
:::info 資訊
內容
:::
```
出來的樣式:
:::info 資訊
內容
:::
```
:::note 筆記
內容
:::
```
出來的樣式:
:::note 筆記
內容
:::

### 圖片引入:

建議您將圖片傳給[discord](https://discord.gg/7g7VE2Sqna)伺服器客服之後，管理員在幫你放上去即可

## 指令選項

指令選項的教學請用以下格式

```
| 選項| 說明|
|-------|------------|
| 選項1      | 說明1    |
| 選項2      | 說明2    |
| 選項3      | 說明3    |
| 選項4      | 說明4    |
```
出來的樣式:

| 選項    | 說明    |
|-------|------------|
| 選項1      | 說明1    |
| 選項2      | 說明2    |
| 選項3      | 說明3    |
| 選項4      | 說明4    |

更多內容可以查看之前編寫完的文檔

## 注意事項

請注意，要換行請直接控一行
例如:
```
內容

內容
```
* 不這樣的話就會變

內容
內容

* 有換行:

內容

內容
